Selected projects and collaborations

Geospatial Map Development Results

Creation of geospatial maps utilizing ArcGIS, QGIS, ENVI, and SNAP software for image correction and spatial information extraction.

Project Details

The workflow began with image data processing in ENVI and SNAP to perform image correction and extract spatial information such as land use, vegetation, and topography.

The processed outputs were then combined and analyzed using ArcGIS and QGIS to produce detailed, thematic, and integrated maps.

Academic Collaboration Activity

Collaboration with Dr. Drs. Harmes, M.T. in a study on land-use change simulation in Jambi City using the Analytic Hierarchy Process (AHP) method.

Project Details

The activity included the following processes:

  • Weighting of driving and inhibiting factors influencing land-use change
  • Scoring and analysis using the Analytic Hierarchy Process (AHP)

Output: Simulated land-use change data projection for the year 2029.

MATLAB Based Geospatial Analysis

Image data analysis using MATLAB for spatial processing, filter application, and data validation through statistical testing methods.

Project Details

Applied image filters and created analytical diagrams to visualize patterns and data characteristics. Filters were used to enhance image quality, reduce noise, and clarify observed objects.

The final stage involved testing and validating the data using MATLAB’s statistical methods to ensure accuracy. The output included geospatial maps and analytical datasets for further interpretation.

Geospatial Analysis Activity

Geospatial analysis using RStudio with COVID-19 and population data from the Central Bureau of Statistics (BPS) to map case distribution in Kuningan Regency.

Project Details

Data Used:

  • Shapefile (SHP) data of Kuningan Regency
  • Population and COVID-19 data from the Central Bureau of Statistics (BPS)

Output: Geospatial data showing the distribution of COVID-19 cases in relation to population density in Kuningan Regency.
